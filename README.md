# CinemaScope_Analytics_Excel_Dashboard

# Movies Data Analysis & Excel Dashboard --

##  Project Overview
This project involves end-to-end data analysis of a movies dataset using Microsoft Excel. 
The objective was to clean the data, perform analytical problem-solving, and build interactive dashboards 
to derive insights related to movie performance, ratings, and director–star collaborations.

---

##  Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Excel Formulas & Functions (Vlookup ,Hlookup ,Xlookup ,Conditional Formating ,etc
- Forecast Sheet
- Data Visualization

---

## Analysis Performed
- Missing data handling and cleaning
- Genre-wise and year-wise analysis
- Budget vs Gross earnings comparison
- IMDb score categorization
- Profitability calculation
- Director–Star network analysis
- Predictive analysis using Excel Forecast Sheet

---

## Dashboards
The project includes a **3-page Excel dashboard**:

1. **Overview**
   - Total Movies
   - Average IMDb Rating
   - Average Budget & Gross
   - Genre and release trends

2. **Financial Performance**
   - Budget vs Gross analysis
   - Profitability by genre
   - Top grossing movies by year
   - Revenue-to-budget ratio trends

3. **Content & Network Insights**
   - IMDb rating trends
   - Rating popularity over time
   - Director–Star collaboration network

##  Key Insights
- High-budget movies generally earn higher gross, though exceptions exist.
- Certain director–star collaborations consistently result in high box-office success.
- PG-13 and R-rated movies dominate recent years.
- Profitability has improved over time with increasing budgets.
